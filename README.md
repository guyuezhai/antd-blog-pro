# antd-blog-pro
a blog use antd react umi dva

```
    git clone https://github.com/guyuezhai/antd-blog-pro.git
    cd antd-blog-pro
    npm install -g cnpm --registry=https://registry.npm.taobao.org
    cnpm i
    npm run dev
```
